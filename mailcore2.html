MailCore

An e-mail library for Mac & iOS

MailCore is a Mac and iOS library designed to ease the pain of working with e-mail protocols. MailCore is batteries included with support for SMTP, IMAP, POP3, MIME and rendering of HTML messages.

Announcing MailCore 2!

From the authors of Sparrow and MailCore 1.0 comes the new and improved MailCore 2! MailCore has been completely redesigned from the ground up with a new blocks based API. In addition to the existing support for IMAP and SMTP, here are some of the new features:

  • UI widgets for HTML message rendering
  • An asynchronous API with Objective-C blocks
  • POP3
  • A portable core engine

Getting the Code

You have two options for setting up MailCore, from simplest to hardest:

CocoaPods

Add 'MailCore2', '~> 2.0.0' to your Podfile. That's it!

Raw Sources

If you are brave and want to build from raw sources then checkout directly from git:

git clone https://github.com/MailCore/MailCore2.git 
cd MailCore/
git checkout 2.0.0 # Checkout the latest stable version

    FAQ

    What happened to MailCore 1.0?

    Fear not, MailCore 1.0 will continue to be maintained as many apps use it. Under the hood both libraries use the same C library libEtPan. New features however will be added to MailCore 2.0. There is a synchronous C++ interface in MailCore 2.0 that would be a suitable replacement for the MailCore 1.0 API. Depending on demand, we may create an Objective-C synchronous API that would make transitioning easier.
